1) Посмотреть где я : pwd
2) Создать папку : mkdir papka_2
3) Зайти в папку : cd papka
4) Создать 3 папки : mkdir dz_1 dz_2 dz_3
5) Зайти в любую папку : cd dz_1
6) Создать 5 файлов (3 txt, 2 json) : touch z_4.txt z_5.txt z_6.txt d_7.json d_8.json
7) Создать 3 папки : mkdir dz_4 dz_5 dz_6
8) Вывести список содержимого папки : ls -la
9) + Открыть любой txt файл : vim z_5.txt
10) + написать туда что-нибудь, любой текст : Нажать i. {“group”: 25}
11) + сохранить и выйти : Нажать esc. Затем : wq
12) Выйти из папки на уровень выше : cd ..
—
13) переместить любые 2 файла, которые вы создали, в любую другую папку : mv d_7.json dz_4/d_7.json
              mv d_8.json dz_4/d_8.json
14) скопировать любые 2 файла, которые вы создали, в любую другую папку : cp dz_1/z_6.txt dz_1/dz_5/z_6.txt
              cp dz_1/z_5.txt dz_1/dz_5/z_5.txt
15) Найти файл по имени : find -name z_6.txt
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает : tail -f z_5.txt
17) вывести несколько первых строк из текстового файла : head -2 z_5.txt
18) вывести несколько последних строк из текстового файла : tail -2 z_5.txt
19) просмотреть содержимое длинного файла (команда less) изучите как она работает : less z_5.txt
20) вывести дату и время : date
